# TravelTrucks 

**Веб-застосунок для оренди кемперів**

Сучасний фронтенд-проєкт для компанії, яка займається орендою кемперів по всій Україні. 

##  Основні функції

- **Головна сторінка** — привабливий банер з закликом до дії
- **Каталог кемперів**:
  - Фільтрація за локацією, типом кузова, двигуном та трансмісією
  - Нескінченне завантаження (Load More) через `useInfiniteQuery`
  - Картки кемперів з обрізаним описом
- **Сторінка деталей кемпера**:
  - Велика галерея зображень з thumbs (Swiper)
  - Повна інформація про кемпер (Vehicle Details)
  - Блок відгуків користувачів
  - Форма бронювання з валідацією та нотифікаціями
- Відкриття сторінки деталей у **новій вкладці**




##  Технології

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS** / CSS Modules
- **TanStack Query (React Query)** — для пагінації та кешування
- **Swiper** — галерея зображень
- **React Hot Toast** — сповіщення
- **React Icons**
